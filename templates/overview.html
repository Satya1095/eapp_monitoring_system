OVERVIEW:

The E-App monitoring system monitors the E-App application. In order to evaluate the performance of E-App, certain performance metrics (user status, number of bytes of data uploaded, number of bytes of data downloaded, number of bytes of data shared with other user) are used . The E-App monitoring system connects with the E-App database and various performance metrics for the user are displayed using graphs.

E-APP:

E-App is a smartphone application for emergency preparedness and response in rural areas. E-App draws data from the Internet (applications such as Facebook and Twitter) onto a centralized server. Then the data is distributed among rural residents with E-App on their phone. This app supports techniques such as phone to phone data offloading. This allows transfer of emergency information from the users who have frequent internet access to the users who are not often online.
